# Money Saving Planner

A modern, lightweight savings planner built with React + Vite + TailwindCSS.

## Features
- 3-step goal creation wizard with progress/loading state
- Daily/weekly/monthly plan selection with auto installment calculation
- Dashboard with progress, remaining amount, days left, and suggested daily saving
- Deposit/withdraw transaction forms with timeline records
- Settings, language toggle (EN/ID), theme toggle (Light/Dark/System)
- Privacy Policy and Terms pages
- LocalStorage persistence using Zustand
